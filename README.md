
  # KLR restyling Andre :)

  This is a code bundle for KLR restyling Andre :). The original project is available at https://www.figma.com/design/qJjnGVaKXpYN5DPFzxihqz/KLR-restyling-Andre---.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Media uploads

  The admin image uploader uses Firebase Storage. Configure these environment variables in Netlify:

  - `FIREBASE_STORAGE_BUCKET`
  - Either `FIREBASE_SERVICE_ACCOUNT` with the full service-account JSON, or `FIREBASE_PROJECT_ID`, `FIREBASE_CLIENT_EMAIL`, and `FIREBASE_PRIVATE_KEY`
